# Todo Telegram Bot

Курсовой проект по дисциплине «Программные системы и автоматизация процессов разработки».

Telegram-бот для управления задачами и напоминаниями.

## Технологии

- Python
- pyTelegramBotAPI
- SQLite

## Возможности

- создание задач;
- просмотр списка задач;
- редактирование задач;
- удаление задач;
- отметка задач как выполненных;
- обработка даты и времени из текстового сообщения.
